# dnd-spell-analyzer
Compare spell damage, credit to GorillaofDestiny on youtube
https://www.youtube.com/watch?v=yrgbi1xPESc

Interactive tool to compare spell damage distributions.

Built with Streamlit.

Inputs:
- Spell Save DC
- Enemy save modifier
- Dice rolls (ndx)

Outputs:
- Damage distribution graphs
- Expected damage table
